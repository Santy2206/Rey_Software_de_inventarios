# Rey_Software_de_inventarios
REY es una solución integral de escritorio diseñada para la administración eficiente de productos, ventas y movimientos entre múltiples bodegas (Fragancias, Bala Negra y General). este sistema garantiza la trazabilidad total de las operaciones y la integridad de los datos en entornos de red local.

🚀 Características Principales
Gestión Multibodega: Control centralizado para las bodegas Fragancias, Bala Negra y General.

Afectación Automática: Registro de ventas con descuento automático del stock en la bodega correspondiente.

Control de Bajas: Módulo específico para reportar productos dañados, caducados o ajustes manuales por faltantes/sobrantes.

Seguridad y Roles: Sistema de autenticación con niveles de acceso diferenciados para Administrador y Vendedor.

Resiliencia de Datos: Operatividad total en red local sin dependencia de internet, con capacidad de sincronización en la nube.

Reportes Avanzados: Generación de informes detallados sobre el valor del inventario, entradas, salidas y ventas.

🛠️ Stack Tecnológico
Lenguaje: Python.

Interfaz (UI): Flet (basado en Flutter) para una experiencia moderna y fluida.

Base de Datos: PostgreSQL (Normalizada) para garantizar concurrencia y consistencia.

Persistencia: SQLAlchemy para el mapeo objeto-relacional (ORM).

Sincronización: Supabase (PostgreSQL en la nube) para respaldos automáticos.

📋 Requerimientos del Sistema
Compatible con Windows y equipos portátiles.

Funciona bajo protocolos de red local para soporte multiusuario.
